# jobberwocky
A service that works as a store for job opportunities, 
where companies can share open positions. 

How use it?
1. Clone repository
2. In terminal:
 - yarn
3. Run the aplication:
 - yarn run start
4. Run api server:
 - yarn run server
5. Run tests:
 - yarn run test
